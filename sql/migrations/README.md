# sql/migrations/

Alterações de estrutura do banco ao longo do tempo. Uma mudança por arquivo, com data no nome (ex.: `2026-06-19-criar-tabela-usuarios.sql`). Não alterar migração já aplicada; criar uma nova para novo ajuste.

O schema inicial completo será definido na etapa de modelagem do banco (`sql/schema.sql`), seguindo `docs/boas-praticas/boas-praticas-banco-dados.md`.
