# Mockups

Esta pasta guarda os mockups visuais de referência do Workspace S&A.

Coloque aqui a imagem gerada a partir do prompt de `../04-prompt-mockup-telas.md`.

Convenção de nome:

```txt
mockup-telas.png
```

Se houver mais de um quadro (ex.: autenticação/dashboard e operação), use nomes claros:

```txt
mockup-telas-auth-dashboard.png
mockup-telas-operacao.png
```

Observações:

- Estes arquivos são referência de design, não assets finais de produção.
- Não mover para `public/assets/`. Assets finais (logo, ícones) só entram em `public/assets/` depois de aprovados.
- Sem acentos e sem espaços nos nomes dos arquivos.
